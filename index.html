Q.No.7 Write a function that accepts parameter n and returns factorial of n

Output :
            
            function factorial(n){
                let ans = 1
                for(let i=2 ; i<=n ; i++){
                    ans *= i;
                }
                console.log(ans)
                return ans;
            }
